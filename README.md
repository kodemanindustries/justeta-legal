# JustETA Legal

Public-facing legal pages for [JustETA](https://justeta.app), served at
**https://kodemanindustries.github.io/justeta-legal/** via GitHub Pages.

The Play Store and App Store privacy declarations link here. Treat the
rendered URL as the canonical version — edits to the Markdown ship to
production as soon as Pages rebuilds (~30 seconds after a push to `main`).
